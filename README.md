# Battery Health Prediction using NASA Dataset

## Overview
A machine learning project to predict battery health using the NASA battery dataset.

## Project Structure
- `data/` - Raw and processed datasets
- `models/` - Trained ML models
- `plots/` - Visualizations
- `generate_data.py` - Data generation
- `preprocess.py` - Data preprocessing
- `train_model.py` - Model training
- `evaluate.py` - Model evaluation
- `visualize.py` - Visualizations
- `predict.py` - Predictions

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Generate data: `python generate_data.py`
3. Preprocess: `python preprocess.py`
4. Train: `python train_model.py`
5. Evaluate: `python evaluate.py`

## Results
- Model: Random Forest
- Dataset: NASA Battery Dataset
